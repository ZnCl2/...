终于又打开了编辑器开始写文（日）章（记）😂。开学之后，一整个学期都没有更新主页的文章（但是写了一个生成“图片字”的网页）。开学之后，想考研的心情迅速减退，考研自习室在写了上一篇文章之后就再也没去过……我认为我不是那种能沉下心学习的人。而且我对考研的研究几乎为零，别人在谈论考什么学校复习什么资料的时候，我的脑子都是空白的……于是我又在学校浑浑噩噩地呆了一学期……

这次学校放假特别晚，20号（2021年7月）才考完最后一科。在考试周，受到此前三周的参观实习和各种小事的影响，我归心似箭，考试的内容都是在考试前夜开始背诵的。幸亏绝大多数老师知道我们的真实实力，老师们在考前或多或少的使我们领会到一些考试内容——除了最后一科的。

食品工艺学。这节课的教师有三位，这三位老师和其他老师一样为我们提供了资料。但是我们强烈的归心和浮躁的状态只能让我们记住其中一位老师提供的内容。考试前一天，我们寝室的学生决定背诵为我们授课时间最长的老师提供的材料。没多久，要背诵的内容就从电脑中“复制”到了纸上和胶带上（人话：打印了背诵版和小抄“水晶条”版）。经过半个晚上的“奋斗”，我们认为已经“万无一失”了，于是我们进入了梦乡……

……最终的考试时刻终于到来了！……！！！我内心中千万只草泥马在奔腾，这都是些什么题，我怎么见都没见过……。幸亏有一道计算题是那份材料上有的……还让我忘了……正当我要拿出小抄的时候，我抬头一看，！似乎监考（两位）、流动监考、学工办主任这四双眼睛从我的眼睛中读取出了什么……监考变得更加严格了……于是我绞尽脑汁往答题卡上“糊”了一些似是而非的文字，然后“洋洋洒洒”地交了卷，带着无限的遗憾走出了考场。

回到寝室一看，考试内容大部分是那位教我们时间最短的老师的材料上的，——而且这材料上只有题目，没有解答……真是一位“不识好歹”的老师……看来今年的暑假时间又要变短了……（开学前补考）。考完了试，我们寝室几个人在小饭馆吃了顿饭，然后开始了回家的行程。

今年的天气比往年更糟，又热又闷，导致回家路上使我最幸福的是火车上的空调……不过这仅有的好心情被一盒无骨鸡爪破坏掉了。出发前在饭馆吃饭，上菜之前我去超市准备路上吃的东西，其中有这一盒花了14元人民币的外包装精美的无骨鸡爪……转车了，我在火车站候车室准备好好品尝一下的时候，突然发现——在这精美的外表之下却是漏气的内包装和发霉的一次性筷子……

看着盒中这堆乱七八糟的东西，心中很不是滋味，但是此时我已经离卖这东西的超市已经几百公里，不可能再坐车回去追究。于是我又带着无限的遗憾和悲哀（？）把这已经发霉的东西扔进了火车站的垃圾桶……

被火车颠了一晚上，终于到了家。家里除了温度太高以外都和平常一样，然后立刻开始刷（结满尿垢的）马桶，洗（五个月不洗的）毛巾，刷（每顿饭都刷但每次都没刷净的）碗等等卫生工作……否则我活不下去。

此次假期依然准备了“假期闲的没事的消遣”——但是是从“某寻梦公司出品的购物软件”上买的。

在家经过了几个“惊心动魄、魂不守舍”的夜晚，教务系统还不能查成绩的时候——学工办主任给我发微信，说我被抓补考了……

没过两天全院的成绩单就出来了，整个班只有我和另一位同学食品工艺学不及格……我一看这分数，……这就是“一道计算题的差距”（又名：一道计算题没抄上引发的惨案）……

但是日子还是得过的。家里非常闷热，家里的电扇几乎整天开着，我整天出汗，似乎有一些作用。……幸亏家里的卫生间可以洗澡，否则我会被我身上干掉的汗水腌透……

也许再过两天我就得去亲戚家“避难”（避暑）了……首先定一个小目标，把食品工艺学的题背牢……

这次文章就写到这里……（文明）的，我上次说的把图片上的字打出来也没打完……看着编辑器上只打了一半的文章，……我也不知道我什么时候能把它全打出来……

如何在Zeronet上创建Mixtape播放列表？

原文Eibriel写于2018年5月7日
TCJY翻译2019年8月25日

Lola正在开发（TCJY：现在似乎停止开发了）的零站“Mixtape”可以让你创建和分享包含封面、标题、艺术家和音频文件的播放列表。
这个零站现在还没有能做这种事的按钮（TCJY：现在已经有了），但是～嘿嘿！自己动手丰衣足食～（TCJY：以下为“土办法”）
要想分享你的Mixtape，你的Zeronet端口得是开放的。
首先查看这个演示用播放列表，你会看到一个空的零网网站，因为它是Mixtape播放列表，不算是一个零站。
然后去ZeroHello找到叫做“Mixtape”的聚合站（播放列表），把鼠标指针放到名称上就会出现“三个点”。点击“三个点”然后选择“克隆”选项，你就会拥有一个空的播放列表！
这个新的零站（播放列表）将被打开，记下网址后面的一串（TCJY：站点地址）（例如：“15k39oW3xh8WCrG2mvgC53EcQisHLAqnox”）。
然后难点来了，打开文件管理器，找到Zeronet的安装文件夹，打开其中的“data”文件夹，你将会看到一吨名字看起来像零网地址的文件夹。（所以说这是难点）
找到和你刚才记下的站点地址一样名字的文件夹，打开。你会看到“data/users”文件夹。移动到“user”数据备份文件夹。
回到网页浏览器来重命名咱刚创建的Mixtape，按住右上角的“0”图标并向左拖。找到“标题”、“介绍”栏做相应的修改，然后点击“保存站点设置”。
然后在“内容发布”选项中选择“content.json”，选择“签名并发布”，再选择“data/users/content.json”，再次点击“签名并发布”（你可能会看到出错的提示，别担心，这是正常现象）。
好了，既然这个Mixtape已经改成我们想要的名字了，那么我们还要让这个零站知道我们是主人。
在刚才的文件管理器打开文件“data/users/content.json”然后修改文件内容使其看起来像下面这样（不要复制粘贴！），忽略掉“some_code”部分，最重要的是不要落下“},”和最下面的几行。 



{
	"address": "some_code",
	"files": {},
	"inner_path": "data/users/content.json",
	"modified": 1525634594,
	"signs": {
		"some_code": "some_code"
	},
	"user_contents": {
		"cert_signers": {
			"zeroid.bit": ["1iD5ZQJMNXu43w1qLB8sfdHVKppVMduGz"]
		},
		"permission_rules": {},
		"permissions": {
		"lola@zeroid.bit": {"max_size": 100000}
		}
	}
}


最后将“lola@zeroid.bit”改成你的Zeroid。
现在你可以回到Mixtape，点击你的Mixtape，然后你就可以看到“Add”（添加）按钮（什么？！你没看到？！那真是见鬼了！去问问文章的原作者吧）。
选一张封面图（分辨率为240×196），填写标题、介绍，然后选择音频文件（浏览器能播的格式，“.mp3”、“.ogg”等）。点击Submit（提交）（又会提示出错，不用管它）。注意：歌曲是反序读取的，后上传的歌曲先播放。
完工！你可能需要在你的播放列表站点点击“签名并发布”（不是Mixtape主站！）发布更新（TCJY：原作者表示不确定，但我确定以及肯定）。
你也可以去看看（TCJY：原作者的）测试零站，这（也许）是宇宙最史诗级的Mixtape：《银河护卫队～第一卷》。

音频处理之仿磁带效果

2019年8月7日

使用软件：GoldWave

一、讲故事篇

我小时候非常淘气，是上课不听课的那种。
有一段时间我坐在最后一排，后排的角落里有一袋（班主任的）（旧）书……
这袋书里很多是《×的生活》，但是最让我印象深刻的是一本介绍“电化教育”的书。
这本书上讲了投影仪、录音机、幻灯机、录像机的用法和在教学中的应用……
……然后我就让我爸买了一个录音机……
后来我用这录音机录制了很多动画片的片头片尾曲——几乎都弄丢了
这之后没几年就到了MP3大行其道的时代……
前段时间我翻出了录音机和磁带……

二、理论篇

磁带录音的时候由于抹音方式的不同会有大小不同的底噪；
（在音频中添加噪音）
录制和播放时会有失真、电流声和爆破声等等；
（添加电流声、爆破声和失真）
磁带机具有机械结构，走带时会产生抖动（jitter）；
（在音频中添加抖动）
录音磁带的音频带宽实际上不大；
（低通滤波）
有些机器在输出音频之前会进行去破音的处理。
（去破音）

三、实战篇

打开音频文件
原音频处理：
——添加电流声
打开工具-运算式生成工具，填入以下运算式
wave(n)+x*abs(sin(2*pi*f*t))
在“常数”里的“x=”填入适当数值（电流声大小）（我使用的值是0.05）
“f=”填入50（60）（交流电频率），确定
——添加失真（有很多方法，这里的方法仅供参考）
打开工具-运算式生成工具，填入以下运算式
sin((pi/2)*sin((pi/2)*wave(n)))，确定
——抖动样本生成：
效果-重新采样（采样率要尽量高，否则处理抖动时会有干扰。）
全选，复制，粘贴为新文件（为了生成和原音频时长相等、采样率相同的抖动样本）
选择新生成的文件，打开工具-运算式生成工具，选择预设-噪声-白噪声，确定
打开效果-滤波器-低通/高通，设置截止频率为20Hz，斜率保持默认，低通，静态，确定
打开效果-立体声-声道混合器，选择预设-单声道混合，确定（左右声道在同一条磁带上，抖动数据相同）
效果-音量-最大化音量，默认设置，确定
——添加抖动
选择原文件，打开工具-运算式生成工具，填入以下运算式
wave((n+x)-x*(waveA(n)))
其中A为生成的抖动样本的文件编号（在运算式框下面的“来源”查看文件号）
在“常数”里的“x=”填入适当数值（点试听，音频应稍有抖动而又不严重变调），点确定
——（在这之后可以关掉抖动样本、将音频重采样回原来的采样率以节省资源）——
——添加噪声、爆破声
打开工具-运算式生成工具，选择预设-噪声-黑胶唱片，默认设置，确定
——低通滤波
打开效果-滤波器-低通/高通，设置截止频率为8000Hz，斜率设为3，低通，静态，确定
——去破音
打开效果-滤波器-破音/喀哒声，容限（误差值）设为750，确定
——（至此处理完成）——
