Fixed My Simple Tor Outproxy For I2P Using DeleGate

by Miracle Max

Posted on June 22, 2018

A while ago, I wrote about using the tiny proxy DeleGate. I mentioned that it seemed to work as an i2p outproxy, but I was seeing missing images and css. Well, I figured out what was happening.

Original Post: Creating an i2P Tor Outproxy with DeleGate

The problem

So it seems that the default tunnel settings for the i2p router are pretty restrictive. It limits the number of requests someone can make per hour/day/etc. I noticed this when I started adding pages to the Book Club on this site and I couldn’t see most of the images.

The fix is to go to the Server Throttling section of the hidden service options in the i2p router. Simple fix, but unless you know that sever throttling is even a thing in i2p, you’re left without much of a clue.

Update 2018-06-23

OK, maybe this wasn’t the fix I thought it was. This may have been a browser-related issue. Going to the Pirate Bay tor site with Midori still shows a bare-bones front-page (search results look normal), while everything looks as expected in other browsers. Oh well. I’m going to keep using i2p with this tor outproxy just because I like it that way. ;-)
