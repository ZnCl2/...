Investment Category Welch’s T-Test P-Values

A Welch’s t-test compares two samples (assumed to have unequal variances) to see how likely it is that they really come from the same population (and therefore the different averages are only due to chance). The number to focus on is the p-value; it tells you the odds that you’re only looking at a chance difference instead of a real one.

I think we can use the p-value as a useful guide. If the p-value for a comparison between two strategies is low (< 20%), then I think it’s reasonable to assume that there probably is a difference between the two strategies and we can prefer the one with the higher CAGR. However, if the p-value is high (> 50%), then it’s probably reasonable to assume that there isn’t much of a difference between the two strategies and we should consider them to be the same as far a total returns go. Between 20% and 50% is sort of a gray area.

Filter Details

All investments are pre-filtered so that only those with an inflation-beating dividend yield, 5-year dividend growth, and 10-year dividend growth are candidates for selection. After that, four general-purpose filters are applied (see first section of the table below). Next, for each Sector/Measurement combination where (1-p)*abs(R) >= 0.475, criteria are automatically select from statistical correlation data.

Using regression analysis, we calculate the minimum or maximum value needed to select inflation-beating investments. We choose the lowest value for the tops and the highest value for the bottoms. NB: Because two types of regression are evaluated for fit (linear and logarithmic plots), some measurements appear twice.
