Name.YO

Name.YO is a centralized name provider, aimed at easy single-click and free domain name reserving. Name.YO supports the following TLDs:

.yo .yu .of
.inf .zn .list

Subdomains (both wild-card and not) are supported. Example domains: name.yo, devtalk.yo, test.yu, best.of, gitcenter.inf, talk.retroshare.zn, shorten.yo, sites.list, nope.list.

Install Name.YO plugin do bring Name.YO address support to ZeroNet:

Register domain

You can register high-level domains (e.g. name.yo), subdomains (e.g. talk.name.yo), or wildcard domains (e.g. *.name.yo). You can only register a subdomain (or a wildcard domain) if you own the parent domain (e.g. you can register talk.name.yo only if you own name.yo).

In case you accidentally use a wrong address, just re-register the same domain again.
