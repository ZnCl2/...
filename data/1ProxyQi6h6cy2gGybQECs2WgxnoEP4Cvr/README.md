# ZeroHello

Homepage for ZeroProxies

ZeroNet address: http://127.0.0.1:43110/1ProxyQi6h6cy2gGybQECs2WgxnoEP4Cvr

## Setup

Start ZeroNet with `python zeronet.py --homepage 1ProxyQi6h6cy2gGybQECs2WgxnoEP4Cvr`

If you want to customize the texts install the plugin.
Either copy it from the Zite (after visiting it) `cp -r data/1ProxyQi6h6cy2gGybQECs2WgxnoEP4Cvr/plugin plugins/HelloZeroProxy` or clone the repo and copy it from here `cp -r HelloZeroProxy/plugin ZeroNet/plugins/HelloZeroProxy`
