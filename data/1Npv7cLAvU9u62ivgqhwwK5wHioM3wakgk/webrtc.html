WebRTC IP Leak VPN / Tor IP Test

WebRTC es un protocolo de comununicación que se basa en JavaScript y que puede filtrar tu dirección IP actual incluso detrás de un VPN. Mientras que software como NoScript previene esto, es probablemente también una buena idea el bloquear este protocolo directamente, sólo para estar a salvo. Esta página comprobará si tu navegador web está afectado por la filtración WebRTC.

Esta demo hace de manera secreta peticiones a servidores STUN, que pueden registrar tus peticiones. Estas peticiones no se muestran en una consola de desarrolladores y no pueden ser bloqueadas por extensiones del navegador como AdBlock, Ghostery, etc.
