Solutions for the refferer error
- Copy the search result link manually and open it in a new tab/window.
- Install the Firefox RefControl add-on and use these settings. (This might break other ZeroNet sites.)

Notes
- This search engine is buggy as hell.
- Only file names of media files and some meta data are currently indexed. Content search doesn't work yet.
- Search results should show up immediately, otherwise something went wrong (e.g. missing files).
- Zearch makes use of the PedoBlocker blacklist.
- Some other ZeroNet search engines (in alphabetic order): Bwoi, Kaffiene, ZeroSearch, (add yours)
