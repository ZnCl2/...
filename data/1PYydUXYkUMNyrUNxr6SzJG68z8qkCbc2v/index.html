COMPRESSION

ZPAQ is similar to ZIP/7-Zip. It offers a widespread of settings and allows an even better compression ratio. The disadvantage is CPU time you need to invest to decompress. It takes days for certain leaks (400GB+) to be compressed and a few hours to be decompressed.

Reason why I took this decision was that I see this project as an archive which therefore rarely is decompressed.

A split is done every 50MB because of how "bad" zeronet big files handles.
