interpretation: ZeroVersions

Should you upgrade, or not?

It depends what kind of user you are. If you:
  • are just a normal user and want a stable version, stay with the releases announced here. Probably you will also like the ZeroBundle.
  • want to run new features earlier and also help reporting bugs/wishes for them, use the master branch from here.
    But keep in mind: things may break, so: always make a backup before!


Where can you find out more about these new features ?
  • commits from code here
  • change log on reddit
  • dev blog
