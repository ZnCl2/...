CLI

Install globally with npm to use the zws command line tool: npm install -g git+http://127.0.0.1:43110/1JKPBG3dLp9FARyarPoR3Mv3p5d9Fy9Em1/zeronet-websocket.git

Usage: zws [options] cmd [... params]

--host, -h

Host of ZeroNet UIServer. Default: 127.0.0.1

--port, -p

Port of ZeroNet UIServer. Default: 43110

--site, -a

Address or domain name of target site. Default: 1HeLLo4uzjaLetFx6NH3PMwFP3qbRbTf3D (ZeroHello)

--secure, -s

Use TLS.
