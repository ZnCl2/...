Usage

File Dependencies

You don't need all JS files to make a simple file downloader! Before you begin, it is very useful to learn about the structure of ZeroMux. Feel free to remove the parts you don't need.

  • utils.js

    This file contains beautiful stateless functions that might be useful in a file downloader. It does not depend on any other files.

  • reader.js

    This file is the JSON reader for the file.json, a file list that important information on how the file chunks should be reassembled. This JS file has no states or dependencies.

  • download.js

    This file is responsible for downloading, verifying and reassembling the big file. It has no states. To use it, you will need all 5 files listed below:

    asmcrypto.js
utils.js
reader.js
async-hash.js
download.js

  • mp4worker.js

    This is part of the MP4 fragmenting tool. To use it, you will need all 4 files listed below:

    mp4parser.js
mp4alg.js
mp4builder.js
mp4worker.js
