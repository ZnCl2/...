Zerocoin (ZNC)

Cryptocurrency assisted by Git (using commits as blocks in blockchain) to help ZeroNet and other projects.
  • Wallet zite
  • Blockchain as merger site
  • Blockchain size
  • Blocks count
  • Sync status and date behind
  • "Unit to show amounts in" selector
  • Fees are for the Open Fund

    CryptoID

    Decentralized unique IDs (blockchain based in Git).

  • Import users from old CryptoID
  • Register
  • Blockchain as merger site
  • Search if user exists in blocks/commits (confirmed by other users)
  • Blockchain size
  • Blocks count
  • Sync status and date behind
  • Show Zerocoin count, with link to ZNC Wallet
  • Verified users in .json (with badge) by owner

    • Assigned to: Daniell

    Addresscoin

    .www domain extension in ZeroNet (blockchain based in Git).

  • Login with ID
  • Moved by Zerocoin
  • Blockchain as merger site
  • Domains are active in blocks count
  • Blockchain size
  • Blocks count
  • Sync status and date behind
  • Show Zerocoin count, with link to ZNC Wallet
  • Users can donate to an address, to domains active (users can choose for which domain they are donating, and the excess is for other domains in the address)
