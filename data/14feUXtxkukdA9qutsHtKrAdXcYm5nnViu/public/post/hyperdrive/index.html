Hyperdrive : Ein P2P Dateisystem

28 Jul, 2021 Filesharing

Hyperdrive ist ein sicheres verteiltes Echtzeit-Dateisystem, das für die einfache P2P-Dateifreigabe entwickelt wurde. Hyperdrive benutzt das Hypercore Protocol.

The “Dat Protocol” has been renamed to the “Hypercore Protocol”

Die wichtigsten Funktionen im Überblik:

  • Versionskontrolliert: Dateien werden standardmäßig versioniert, sodass historische Änderungen leicht zu erkennen sind und Datenverluste vermieden werden.
  • Composable: Mittels dem Mount-System können Hyperdrives in andere Hyperdrives verschachtelt werden, was leistungsstarke Tools für die Zusammenarbeit mehrerer Benutzer ermöglicht.
  • Mit einem Link gemeinsam nutzbar: So kann man ein gesamtes Hyperdrive mit anderen Nutzern teilen, indem man ihnen einen einzelnen 32-Byte-Schlüssel sendet, wie z.B. hyper://9dc6d5ce001b52a461b54b5b8e2238a09dfa226a24051bc25a5dfa09ecc92a38/
  • Sparse Downloading: Standardmäßig laden Leser nur die Teile der Dateien herunter, die sie benötigen, und zwar bei Bedarf. So ist eas möglich Medieninhalte von Freunden zu streamen.
  • Schnelle Lookups: Datei-Metadaten werden in einer verteilten Baum-Struktur gespeichert, was bedeutet, dass Dateien mit minimalen Netzwerk-Lookups gefunden werden können.
  • Versions-Tagging: Man darf Hyperdrive-Versionen x- beliebige Zeichenfolgen oder Namen zuweisen und diese auf dem Laufwerk speichern, so lässt sich problemlos zwischen semantisch aussagekräftigen Versionen unterscheiden.
