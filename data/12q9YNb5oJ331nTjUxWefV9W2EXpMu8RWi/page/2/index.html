    Voluntary decentralized network and blockchain / 自愿性网络和资本主义区块链

    Networks like BitTorrent and IPFS, are voluntary, which means you are not directly paid with anything.

    This sounds like communism, but it also makes rare resources hard to download.

    Though there are improvements to prioritize rare files and and punish suspicious peers, what they can do is still limited.

    Blockchain also came out at that time, and today there are many further applications like FileCoin.

    Read the whitepaper, and you will find that FileCoin is based on PoS.

    So more disk space you own, the more probability you are selected to verify a block and earn money.

    Looks like capitalism, but we have no any other choice.

    In this way, we can almost incentive anything, and bring capitalism to decentralized network, lol.

    As you can see, Bitcoin is far more well-known than any decentralized network.

    Decentralized network with private ownership against copyright and censorship

    诸如BitTorrent和IPFS，节点都是自愿的，意味着你并没有直接获得利益。

    有点像共产主义，但这种做法让稀缺资源无法下载。

    目前有优化算法，提高稀缺资源的优先级，效果有限。

    同时，区块链火了起来。现在终是有了一些实用的区块链算法，如FileCoin。

    读下FileCoin的白皮书，你会发现FileCoin是基于PoS的。

    这又变成资本主义了，但我们没有办法。

    以这种方式，可以激励任何东西，资本主义嘛。

    也造成了比特币很火，自愿性网络一片萧条的景象。

    通俗地解释 Web3.0

    Web3.0是什么东西，这个其实可以分成两种，第一种是以区块链（啧啧，又是炒作）为首的网络，第二种就是像ZeroNet这种，可以自由分享文件的网络。

    那么我要说的是，Web3.0纯粹是一个各家都炒作的术语。

    区块链

    区块链要从比特币讲起，这个币呢，只能作为一种看起来高级点的钱用，但区块链的价值不止于此。

    比特币只能当货币来用，因此有些人就想了，区块链就这点用途么？于是就有了以太坊。

    说白了，以太坊就是更加复杂的比特币，充分发挥已有的东西。

    哦，对了，比特币支持对一笔交易的验证，内置一个脚本判断这笔交易是否成立，这点也是区块链的好处。

    以太坊就更有意思了，加入了虚拟机，把区块链玩出花来了。这样这链上还能执行程序，举一个简单的例子，你转入钱到一个账号，要是超过10块钱，就全部退回来。要执行这个程序不简单，因为要分布式嘛。

    分布式

    分布式就是去中心化，这不废话嘛。

    不，去中心化就没人可以控制了，简而言之，多数人暴政。

    微软的更新、迅雷、BT、微力同步之流都是去中心化的应用，只是铜臭味很浓。

    国家挡不住你下小电影，这就是去中心化的抗审查性。

    下载的速度很快，人越多越快，这是，呸，速度快。

    而且不会因为DDOS，或者什么攻击而瘫痪。

    显然，区块链也是去中心化的，就是为了那个抗审查性。

    这样交易就不会被人控制了，因此经常大涨大跌。

    区块链的高级应用

    比特币都弱爆了，那可不是区块链的精髓所在，又是挖矿的，又是被攻击，浪费能源。

    因此就涌现一批各种币，比如FileCoin。

    区块链就是价值的传递，价值的传递又是基于共识，就是说所有人都统一的，没有偏差的认为某人有几元几角。

    FileCoin是这样的：

    我付钱给你，你帮我存一个文件。文件发给你。

    随后每个月付个几块钱，顺便检查一下你有没有把文件丢了，丢了的话，后面的钱就不给你了。

    有时我想把文件下载回来，给其他网速快的人一点钱，让他帮我下。

    这就是价值的传递。按这种模式，可以做出很多web3.0应用。

    Steemit

    还有Steemit：

    基于区块链的社交平台，奖励获取赞数最多的文章，即激励。

    不含区块链的网络

    全球币圈都乌烟瘴气的，其实我挺讨厌的，大多数没创新的币都是新型传销。

    Web3绝对不是除了区块链就没东西了，还有两个领域，数据传输和匿名网络，注意，都是分布式的，即

    人人为我，我为人人

    数据传输是Web3的基石，匿名网络增强了抗审查性。

    ZeroNet

    名字的意思应该是，一切重零开始，构建分布式网络。倒还真的是有那种零几年互联网的感觉，时空错乱了都。

    不多说。

    • 没人能删帖的论坛，想说什么说什么
    • 盗版问题？没人管得了
    • 你懂的。

    IPFS

    大概是近几年新出来的网络，相信你已经听过这个名字了，被自媒体炒上天了，不是吗？

    抛去一切不讲，你看它有什么用，这就是一个文件系统，一个超大的硬盘。

    你从中国一个电脑上存一个文件进去，电脑别关，大洋彼岸再用这个文件名下载文件，没问题。

    中间有很多复杂的步骤，建立起两个电脑的连接。总之，就是一个分布式的文件系统。

    IPZN

    这个是我目前在开发的网络，结合Web3.0的三个要素，价值、数据和隐私。

    运作原理和零网一样，只不过底层更加高效，选用了IPFS。

    在此之上，我又会构建一种新的币（停停停，不是所有币都是垃圾币），用来激励搜索服务，参考FileCoin和Steemit的运作机制。

    创造一个真正实用的去中心化搜索引擎，意味着Web3将得到极大的发展，毕竟，现在Web3还没有一个能用的搜索引擎呢。

    DWeb search engine

    Two ideas on decentralized search engine

    Several months ago, I wrote a search engine called Horizon, actually it consists of a centralized one and a decentralized one.

    That decentralized Horizon is just a ui and the database which contains the data I crawled from zeroent, but for there’s no enough space to store all the content, I used AI to derive keywords from the content.

    I also added a feature searching things in downloaded sites, this is what other search engines of ZeroNet do.

    Finally the engine is done, but I’m not satisfied with that. So I continue to build a centralized search engine with Elasticsearch.

    Since it’s expensive to run a elasticsearch engine, I stopped making any search engine.

    Recently, I came up with IPZN, the ZeroNet runs over IPFS.

    Search engines are still necessary for dWeb, but building a decentralized search engine is not easy.

    I have been discovering about blockchain, in IPZN, it can be used to:

    • Pay for some service
    • Deposit token (coin) and prevent spam
    • DAO, voting system

    Yeah, combined with blockchain, but optional

    With the stuff above, we can have two types of decentralized search engine：

    1. Tagging system, a site for tagging everything on IPZN, or even the whole IPFS.
    2. Searching service market, you pay for a search

    For 2, we can create a token/coin, which is supplied with advertising, so users don’t need to directly pay for it.

    When a user searchs, the request will be broadcasted to all available searching nodes. To prevent DDOS, user should deposit some tokens.

    User can upvote for search results, and the result are mixed with the results from multiple nodes, ranking by the reputation(user upvote) of each node. Refer to existing project Steemit

    So, searching can be either treated as market or steemit

    Still confused how to implement it, seems not robust

    Edit: 8/23

    Searching is different from storing file or what.

    Our goal is to incentive better quality of search results, but the quality is determined by users, real human beings.

    It causes a problem, how to deal with spam and bots.

    For bots, we need a decentralized verifying system.

    And limit maximum operations per hour a user can perform.

    Decentralized verifying system

    DAO and the verifiers in proof of stake mechanism give the requester a challenge (e.g. computer generated captcha), and the verifiers vote to determine accept him or not.

    Edit: 8/24

    Tagging system can be fullly decentralized by using peer reputation and a better sorting algorithm.

    Edit: 8/25

    To be more decentralized, we will use DPoS (Delegated Proof of Stake).

    The new coin can be called SearchCoin, or other better name. It’s just like FileCoin, which incentivizes peers to store files
