Hardware #

  • Intel NUC

    Compact. Inexpensive. Linux friendly.

  • HP Z22n G2 Display

    Compact. Accurate color. 1920 x 1080 is sufficient for Linux. (I use a 4K display with a MacBook Pro at office.)

  • IKBC C87

    • Wired: Bluetooth keyboards may have troubles when booting a full-disk encryption Linux.
    • 80%: Less space but all functional keys (F1-F12, arrow keys, etc) are available.
    • Cherry MX Brown: Cherry MX Switches are reliable. Actually I am fine with both brown and red switches.
    • PBT: Resist to oil, sweat, and lotion. More durable.
    • Steel board: I dislike this. It makes the keyboard heavier.
    • Not programmable: Less convenient but acceptable.

  • Razer DeathAdder Left-handed Edition

    One of the few left-handed mouses available in my region.

Software #

  • i3

    i3 is an active maintained tiling window manager with good documentation. Also its container tree based layout is very flexible.

  • Firefox

    Firefox sucks. But other browsers suck more.

  • vscode

    Mostly used for markdown editing and scripting. Visual Studio Code is non free. I use VSCodium, a community-driven, freely-licensed binary distribution.

  • IntelliJ Idea

    One of the only two software subscriptions I have. I use it for TypeScript, JavaScript, Go, Java, Python and PHP.

  • Leafpad

    Simple.

  • Konsole

    The default terminal emulator in KDE. Quite feature-rich.

  • Dolphin

    Probably the most functional GUI file manager in Linux.
