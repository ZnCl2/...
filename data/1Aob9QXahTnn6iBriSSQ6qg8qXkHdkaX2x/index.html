ZeroNet Template

This project provides a template for creating and developing ZeroNet sites.

It supports NPM dependencies and it uses Gulp and Browserify for bundling scripts. It also supports SASS and automatic deployments with Travis CI.

This repository is mirrored from GitHub. To use it, see details in develop branch. Note that continuous deployment will only work with Travis CI and GitHub.

This repository is a mirror, so please don't submit issues and pull requests here. Use GitHub instead. In the future, you might be able to do this if Git Center will enable better sync with GitHub.
