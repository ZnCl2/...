Q: Will dıgıverse be pay to win?

Never. The game will give players no gameplay reward for donating. Only a cosmetic button, or change of player skin. Which is funny because this game is primarily text only.

Q: How will the dıgıverse economy work anyway?

The game economy will be based on the number of players within the digiverse. Each time a player enters the game the maximum amount of currency on the server will be increased by 10,000. All players will start with 0 currency. Players will earn currency initially by killing monsters. A monster will drop a number of currency equal to its point value. This will circulate through the economy. This currency is a required component to craft equipment (not necessarily excluding other items). This is how the currency will be removed from the economy. If there is no currency to give out, it will no longer be given by killing monsters, and more must be removed from the economy before it can pay out again.
