关于精子

2016 7 月 3 日

精子真名叫「王子亭」，生于 1995.11.25, 20 岁，英文 ID 是 jysperm.

  • 精子是一名 独立博客 作者。
  • 精子创建了 Atom 中文社区。
  • 精子是 皮蛋 和 豆腐 的主人。
  • 精子在 Github 上积极地参与开源社区
  • 精子有 很多粉丝，还有一个 粉丝团主页
  • 精子曾创建了 RP 主机 和 GreenShadow.
  • 精子在高中退学，进入互联网行业工作，目前在 LeanCloud 任 Node.js 服务器端开发工程师。

价值观

  • 精子相信 认真做事的人终有回报
  • 精子认为 过程正义比结果正确更加重要
  • 精子相信科学是最能够提高人类生产力的一套理论
  • 精子相信世界总是在向好的方向发展，不喜欢阴谋论
  • 精子是个很固执，很天真的人，他相信每个人都有像他一样天真的一面

兴趣爱好

  • 精子在无聊的时候喜欢刷 知乎
  • 精子喜欢 看书，精子推荐大家看 这些书
  • 精子喜欢 哆啦 A 梦、皮蛋豆腐 等各种萌萌的东西
  • 精子喜欢睡觉睡到自然醒，喝咖啡，喝牛奶，吃羊肉火锅
  • 精子也喜欢玩游戏：风暴英雄、星际争霸Ⅱ，以及 Steam 上一些独立游戏

观点

  • 精子 不喜欢黑客圈
  • 精子不喜欢二次元圈
  • 精子喜欢 标准化的世界
  • 精子觉得 学校不是学习的好地方
  • 精子一向认为 代码中不应当写注释
  • 精子觉得 比特币是一个很有趣的东西，亦 长期持有比特币
  • 精子 曾经使用过 PHP, 但现在 不喜欢 PHP, 并且认为 世界上最好的编程语言 绝不是 PHP

博客内容

这个博客创建于精子 7 岁第一天上学的时候，之后的十几年里精子的成长和价值观的改变是很大的。
所有的日志发布之后除了错别字都不会再修改，因此他可能并不完全同意早期日志中的观点。
