Поиск ресурсов по ключам

Этот раздел каталога предназначен для поиска ресурсов по общим признакам с помощью ключей. Ключ состоит из трёх латинских букв, заключённых в обратные апострофы (обратный апостроф ставится клавишей Ё в английской раскладке). Например, если ввести ключ `blg` в поисковое поле, страница выдачи результатов покажет только блоги.

  • `blg` — личные и тематические блоги, онлайн-дневники и т. п.
  • `for` — тематические форумы и форумы-флудилки.
  • `brd` — имиджборды, текстборды, оверчаны.
  • `cht` — текстовые чаты.
  • `soc` — социальные сети.
  • `hzm` — хабы ZeroMe
  • `int` — различные интерактивные ресурсы.
  • `gms` — игры (HTML5, JavaScript).
  • `gmr` — ресурсы, посвящённые играм.
  • `sit` — личные и тематические сайты пользователей.
  • `sig` — сайты-галереи (фото, видео, музыка).
  • `sir` — другие различные сайты.
  • `vid` — видеохостинги (ресурсы для размещения видео).
  • `mus` — музыкальные хостинги (ресурсы для размещения музыки).
  • `img` — фотохостинги (ресурсы для размещения изображений).
  • `hos` — другие различные хостинги.
  • `fdw` — файлообменные ресурсы, шаринг, варезники и т. п.
  • `prn` — ресурсы с порнографией.
  • `nws` — новостные ресурсы.
  • `src` — поисковые системы.
  • `cat` — каталоги разной тематики.
  • `gui` — руководства и справочные ресурсы разной тематики.
  • `lbr` — книжные библиотеки.
  • `ser` — сервисы для различных видов деятельности.
  • `eml` — электронная почта.
  • `zns` — инструментарий ZeroNet.
  • `oth` — разное.

Ключи `blg` и `for` могут иметь дополнительные параметры YY (где YY — последние цифры года): `blgYY` и `forYY` — показать блоги или форумы, созданные в определённом году. Примеры:

  • `blg16` — показать блоги, созданные в 2016 году.
  • `for17` — показать форумы, созданные в 2017 году.

Для поиска ресурсов, созданных определённым пользователем, предназначен ключ NICK@ (где NICK — никнейм пользователя). Пример:

  • pupkin@ — показать ресурсы, созданные пользователем под ником pupkin.

Для просмотра всех ресурсов в каталоге введите обратный слэш — /.
