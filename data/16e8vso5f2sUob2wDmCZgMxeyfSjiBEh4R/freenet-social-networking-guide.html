Freenet 社交网络指南(中文版)

本教程翻译自《Freenet Social Networking Guide》,通过六个简单的教程，用图文并茂的形式，一步一步教 Freenet 新手掌握 Freenet 的基本配置，并学会使用一些常见的社交网络工具。

  1. 安装和配置自由网
  2. 建立匿名身份标识（Identity）
  3. 发邮件（Freemail）
  4. 发微博（Sone）
  5. 逛论坛（FMS）
  6. 即时聊天（FLIP）

第一步：安装和配置 Freenet 节点

开始之前，请确保你已经安装了可靠的网页浏览器和邮件客户端。本文使用的是 Firefox 和 Thunderbird。

译者注： 1. 我们暂时以在 Windows 操作系统下使用官网的安装包为例； 2. 由于直接引用原文的图片，因此界面为英文，实际上 Freenet 自带中文界面.

Microsoft Windows操作系统下 Freenet 的安装

一、下载并运行 Freenet 安装包。

安装程序首先会检查 Java 版本。如有需要，你可以立即更新 Java ，或者如果你的 Java 版本足够新的话，你可以直接跳到第4步。

二、默认目录就可以。

三、对广告软件说不。

四、勾选 Additional settings 下的所有选项，并点击 Install 。

五、如果弹出一条 Windows 防火墙信息，请点击 Unlock 。

六、点击 OK 退出安装程序并开始配置。

Freenet 配置

一、你现在应该可以看到配置向导。点击 Choose custom security 。

二、浏览自由网的时候，你应该始终使用浏览器的无痕浏览或隐身浏览模式。如果你尚未这么做，请赶紧切换过来。记得复制URL并重新粘到浏览器的地址栏（译者注：这样可以保持在当前浏览的页面）。

三、点击“Next”。

四、除非你有理由禁用UPnP，否则点击“Next”。

五、这里选择第二项（连接到陌生人）。如果你认识一些自由网用户，你将来可以升级到暗网模式，但是现在，为了接入网络你需要连接到一些陌生人。

六、这一步选择“Normal”。

七、这一步选择“LOW”，但是如果你已经启用了全盘加密，那么这里你可以选择“NONE”。不幸的是，如果你使用一些后面我们将会介绍的社交网络插件，那么更高的安全等级也不会完全保护你的标识。你应该使用一款全盘加密产品例如 Truecrypt ，至于如何安装和设置它则不在本文的讨论范围之内。

八、这里安装程序会推荐一个默认值，基于你的硬盘容量。你可以使用这个推荐的默认值，或者选择其他数值。

九、这一步你有两个选项。如果你想基于包月宽带的限制设置流量限制，请点击“Yes”。如果无需限制，则点击“No”并跳到第十一步。（译者注：如果你使用3G或4G上网卡，每月的流量有限制，此处请谨慎选择）

十、选择流量限制上限，或者手动输入。点击“Select”后会跳到第十二步。

十一、根据你的宽带类型选择正确的流量限制。点击“Next”。

十二、你的自由网节点已经在运行了。在开始安装任何我们即将介绍的社交网络插件之前，你应该检查你的节点允许使用的最大内存值。

十三、进入“Configuration”菜单下的“Configuration”页面。

十四、基于你的电脑内存容量，自由网安装程序选择了一个“最大内存用量”数值。大多数情况下，这个值对于后面将要安装的插件显得太小。请确保这个值大于512MB，这样你的节点才不会超出内存限制。如果这里的值太小，就会经常遇到节点崩溃和重启。

如果你是根据本指南的说明一步一步来的，但是在这里却看不到WRAPPER区域，这是一个需要上报的bug。一个解决方法是，在你的电脑中搜索“wrapper.conf”这个文件，并使用文本编辑器打开它（为了成功定位到这个文件，你可能要在隐藏文件夹和系统文件夹中搜索）。找到以“wrapper.java.maxmemory”开始的这一行，确保这里的值是512，最后保存文件并重启节点。

十五、上述步骤完成后，鼠标滚动到页面最下方并点击“Apply”。

十六、返回节点首页以便找到重启选项。

十七、重启节点以便更新最大内存设置。

十八、确定重启操作。

十九、请等待直到节点完成重载，以便继续下一个步骤。

当你准备就绪，让我们开始设置Web of Trust plugin插件。

第二步：使用WebOfTrust创建匿名标识

WebOfTrust插件（WoT）为众多联络插件解决了在一个无法审查的媒介中出现的垃圾信息问题。

启用WoT的插件使用一个分布式信任系统，以便选择从那些标识下载信息。当你或其他人在你的信任网络中标记一个垃圾信息发送者，对应的标识的分数会掉。当该标识的分数低到一定程度，你的节点和你的信任网络中的其他人将不会从该节点下载信息。

这样一来，当垃圾信息发送这一行为无法被禁止的时候，人们可以选择将相应的节点排斥在信任网络之外。这也是为什么每个标识都有自己的信任列表，这是用来在适当的时候标记其他标识。

由于WoT是许多其他插件的前提，它将会最早被安装。

一、从你节点的首页，进入配置菜单下的插件栏目。

二、载入官方WebOfTrust插件。

三、当插件下载完成后，点击Community菜单下的Own anonymous identities选项。

四、Freenet会自动为你的新标识生成一个随机的名称。你可以刷新这个页面以便得到一个新的标识，尽管如此，为了最大程度的匿名性，你最好还是使用第一次生成的这个。如果你创建了很多标识，只选择了一个听起来不错的名称，那么每个名称都会创建一个图案，这样你可以很容易地和不同的标识联系起来。

当你选择好了比较满意的名称之后，点击“创建”。

五、两个文本框，里面有一段很长的随机符号，这其实是你的公钥和私钥。请花点时间把这些信息复制到一个文件中，并保存到一个安全的地方，万一你的节点出点什么状况就能派上用场。有条件的话，请对这个文件进行加密，以避免被其他人发现你的密钥。

如果你这一步没有复制两段URI，而是想在后面再进行备份，请点击Own anonymous identities页面中的标识后面的“编辑”按钮。

六、成功创建标识后，返回Own anonymous identities页面。

七、能够让其他人看到你的帖子的最后一个必需步骤是介绍标识。

八、不幸的是，你需要等待一些时间以便让WoT插件下载介绍难题。一两个小时后你会发现一些难题可供解答，但是为了最好的性能，请让节点运行一整夜以便下载标识，然后第二天再回到这个步骤。

你也可以跳过介绍这一步并继续设置Freemail和Sone，但是记得一定要回来进行介绍，否则其他人将看不到你发的任何帖子。不等待WoT完成引导也会带来另一个不良后果是，那就是如果Sone也在进行引导时，花的时间就会更长。

九、在等待较长的一段时间以便WoT下载标识之后，你可以返回介绍页面。现在你应该可以看到页面上排满了难题等待你解答。按照屏幕上的指示操作并点击“提交”。

十、在你解决了答曰10个难题之后，接下来事情就是等待其他标识认出你。当信任者的数量变得比零大，说明你的标识被成功地介绍出去了。你应该试着得到至少五个信任者，所以如果有必要的话，请重复介绍流程。

当你的标识被成功介绍出去，下一步就是获得一个Freemail帐号。

Step 5: Joining discussion forums with FMS

If you’ve been paying attention to the menu items in your node interface you will have seen mention of Freetalk as the official forum system.

Don’t install Freetalk.

Freetalk is not ready for use. It doesn’t work very well yet and consumes a lot of resources. A leading cause of people complaining that Freenet runs bogs down their computer is that they have installed Freetalk.

Instead, follow these instructions to install FMS:

Unlike the other components we have installed so far, FMS works best as a separate application instead of a node plugin. You need to go to the FMS Freesite and download the correct version for your system.

Step 1

The content filter won’t allow you to display this file through the browser by default so save it to the Freenet download directory. Make sure to uncheck “Filter data”.

Step 2

Run the installer.

Step 3

You can choose any installation directory you want but the default is probably best.

Step 4

Make sure to install the FMS service.

Step 5

This is the main interface for FMS you may want to bookmark it on your browser.

Go to the Local Identities page and then switch tabs back to the Freenet interface.

Step 6

At this point we’re going to link your WoT identity to your FMS identity so that posts in both places will come from the same identity.

Step 7

Open the Own anonymous identities page.

Step 8

Click on the edit tab to get your keys. You should also have this information backed up somewhere safe after performing step 5 of the WebOfTrust tutorial. If you didn’t do that before, do it now.

Step 9

FMS identity keys are compatible with WoT identity keys except they are written as SSK@ instead of USK@ and the FMS identity does not include the part including and after /WebOfTrust/

Using the following code as a template translate your WoT keys into FMS keys and save the resulting file as identities.xml. Copy your name exactly as it is shown on the Web Of Trust identity page.

<![CDATA[NAME]]> SSK@TRANSLATED_REQUEST_URI/ SSK@TRANSLATED_INSERT_URI/ false false false false

Make sure the format of your keys looks like the example below. They should both start with “SSK@” and end in “,AQECAAE/” (private key) and “,AQACAAE/” (public key).
