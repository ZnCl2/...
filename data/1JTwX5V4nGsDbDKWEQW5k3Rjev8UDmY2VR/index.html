Kiwipedia template

This site is a template for Kiwipedia, a clone of Wikipedia in ZeroNet.
To create a new hub from this template, visit Kiwipedia and choose Don't see you language in list? in sidebar.

If this is your site, and you've just created a new hub, return to Kiwipedia via the following link: init-hub.
