What’s with the colors? Synesthesia Highlighting

Adults can be trained for synesthesia and it can improve IQ by over 10 points1. While syntax highlighting has been shown to improve completion time for short programming projects for people new to programming 2, it has not been shown to increase general IQ.

Like any new thing Synesthesia Highlighting can take a while to get used to, but once one is used to it then it can significantly boost programming performance, as it is akin to syntax highlighting for pretty much all the words in your computer program.

One of the noticeable ways that syntax highlighting helps is that it helps with avoiding typos, since human spike neurons naturally pay attention to the things at the begining of a list and at the end more, the letters in the middle can get jumbled and it is hard to notice. But with synesthesia highlighting it is much easier to see all the letters of a word, and see that a word is miscolored (after you have enough experience with what a correctly spelled word should look like).

In theory it should be possible to have a synthesis of both synesthesia highlighting for the letters, and have a background highlighting for the syntax itself, however vim (my prefered text editor) does not have support for multiple layers of syntax highlighting. So the choice was between syntax highlighting that helps for inexperienced programmers, and synesthesia highlighting which boosts IQ and can improve the overall program regardless of programming experience.
