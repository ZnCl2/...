# Hình ảnh [cấu hình phù hợp với Computer yếu]

Xuật hiện 1 tab như sau Chỉnh hình ảnh ( Chọn plugin và nhấn Chỉnh như hình dưới ) ở phần chọn/Chỉnh plugin đồ họa, bạn tùy chọn như hình (Pete’s OpenGL Driver 1.76) xem hình..
