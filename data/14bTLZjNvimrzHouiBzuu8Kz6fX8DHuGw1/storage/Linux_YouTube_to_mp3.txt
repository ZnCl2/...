Convert YouTube Videos to mp3 with two linux tools

YouTube-DL (A Command Line Video Download Tool for Linux)
    --YouTubeDL Installation--
    sudo add-apt-repository ppa:nilarimogard/webupd8
    sudo apt-get update
    sudo apt-get install youtube-dl

    --Example Command to get the format types--
    youtube-dl --list-formats https://youtu.be/L999dsuQ1Ao

        Example Output:
        17           3gp        176x144    small , mp4v.20.3, mp4a.40.2@ 24k
        36           3gp        320x180    small , mp4v.20.3, mp4a.40.2
        43           webm       640x360    medium , vp8.0, vorbis@128k
        18           mp4        640x360    medium , avc1.42001E, mp4a.40.2@ 96k (best)

    --Example Command to get the desired format--
    youtube-dl -f 18 https://youtu.be/L999dsuQ1Ao

        Example Output:

        [download]  49.1% of 142.60MiB at  3.12MiB/s ETA 00:23^C


FFmpeg (mp3 converter)
    --FFmpeg Installation--
    sudo add-apt-repository ppa:mc3man/trusty-media
    sudo apt-get update
    sudo apt-get install ffmpeg

    --Example Command--
    ffmpeg -i '/home/dailyuser/►1 HOUR◄ Epic Dubstep Music Mix for Gaming 2014 [Dj Alien King]-L999dsuQ1Ao.mp4' -f mp3 -ab 192000 -vn '/home/dailyuser/►1 HOUR◄ Epic Dubstep Music Mix for Gaming 2014 [Dj Alien King]-L999dsuQ1Ao.mp3'
