I'm a game programmer who has been working on a game library for the Java language. This zite is to act as a resource for other programmers who might be thinking about exploring the code behind rendering and game creation. When I began this journey I scraped together the pieces from all over the internet to build what I currently have. It hasn't been easy to find the right algorithms that work well in Java, since there's no resource I could find that's dedicated to Java game creation.

The goal of this zite is to be the resource I was looking for back when I started coding for games.
