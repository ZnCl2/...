[15/01/2016] Opening forum and website

Even if the game isn't available yet, content will be added to the Encyclopedia to explain what the game is about (with advanced stuffs). Players can create a lot of things in-game, following the game rules. A game need to explain the rules to be enjoyable, thus the difficulty isn't given by esoteric statements. So the Encyclopedia will probably show gameplay equations.

The most important part of SU is also the world creation, so there will be tutorials for advanced users if you want to contribute to the universe by creating your own world. The forum is also a good place to help each others with issues and community tutorials (also user for RP discussions).

The greenlight is still waiting for approval, so don't hesitate to vote for it if you want to support the project.
