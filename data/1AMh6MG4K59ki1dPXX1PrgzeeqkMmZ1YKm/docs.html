ZeroCDN Developer Documentation

Introduction

ZeroCDN is a tool and ZeroNet site that allows you quick access to many JavaScript and CSS libraries, all while keeping your site size down. It has a strange way of including libraries, but you can get used to it

Libraries

You will include libraries using their name and version. Here is a list of all libraries currently on ZeroCDN:

Name Version CSS/JS
jQuery 2.1.4 JS
Bootstrap 3.3.5 JS & CSS
D3.js 3.5.6 JS
Superagent 0.18.0 JS

Including libraries

JavaScript

  1. Copy zerocdn.js to your site’s root

  2. Put this in each of your .html files:

    <script src="zerocdn.min.js"></script>
<script>
    ZeroCDN.js("library name", "library version", function() {
        // The library is now loaded.
    });
</script>

CSS

  1. Copy zerocdn.js to your site’s root

  2. Put this in each of your .html files:

    <script src="zerocdn.min.js"></script>
<script>
    ZeroCDN.css("library name", "library version", function() {
        // The library is now loaded.
    });
</script>

Function documentation

ZeroCDN.js(name, version, callback)

Includes a JavaScript library from ZeroCDN

`name`
`String` - The name of the JavaScript library you’d like to import
`version`
`String` - The version of the JavaScript library you’d like to import
`callback`
`Function` - The function to call after the library is loaded

ZeroCDN.css(name, version, callback)

Includes a CSS library from ZeroCDN

`name`
`String` - The name of the CSS library you’d like to import
`version`
`String` - The version of the CSS library you’d like to import
`callback`
`Function` - The function to call after the library is loaded
