Git Center

Кстати, столкнулся с небольшой проблемой юзабилити Git Center-а.

Ссылка на репозиторий выглядит, например, так:

http://127.0.0.1:43110/1FEvdSRAqTEuGpc31s9ad7rPc4jqRpeWfG

При её открытии делается редирект на Git Center-овский GUI репозитория:

http://127.0.0.1:43110/1GitLiXB6t5r8vuU2zC6a8GYj9ME6HMQ4t/repo/?1FEvdSRAqTEuGpc31s9ad7rPc4jqRpeWfG

Не сразу очевидная проблема заключается в том, что с таким подходом мы не можем получить доступ к боковой панели ZeroNet сайта-репозитория 1FEvdSRAqTEuGpc31s9ad7rPc4jqRpeWfG. А некоторые действия можно произвести только через эту панель. Например:

  • Посмотреть, какие именно файлы не засинхронизировались.
  • Переключить флажок This is my site.
  • ZeroHello не даёт возможность удалить owned site, это можно сделать только из боковой панели сайта.

UPDATE:

Как подсказал @gitcenter, для решения этого затруднения достаточно дописать в URL несуществующую страницу:

http://127.0.0.1:43110/1FEvdSRAqTEuGpc31s9ad7rPc4jqRpeWfG/admin

Сайт вернет 404, и при этом можно открыть боковую панель.
