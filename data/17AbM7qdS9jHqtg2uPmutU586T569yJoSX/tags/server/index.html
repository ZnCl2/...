Cyberwald Homepage

July 22, 2017
Es mag nicht sofort auffallen aber unsere CyberWald Homepage unterscheidet sich stark von vielen anderen Seiten im Internet. Die CyberWald Homepage wird über das “Inter Planetary File System” (IPFS) gehostet. Das bedeutet, dass unsere Homepage nicht auf einem zentralen Server liegt, sondern über ein Peer-to-Peer-Netzwerk verteilt wird. Bei einem P-2-P-Netzwerk tauschen die teilnehmenden Rechner die Daten direkt aus und brauchen dafür keinen zentralen Server. Ruft man unsere Homepage über das IPFS-Netzwerk auf dann kann die Webseite von überall auf der Welt übertragen werden.
