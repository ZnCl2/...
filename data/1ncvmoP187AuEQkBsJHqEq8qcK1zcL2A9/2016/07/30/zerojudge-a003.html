ZeroJudge: a003 兩光法師占卜術

Jul 30, 2016 • Jun 2, 2018 • WillyPillow • en, zh, bilingual, zerojudge, c, competitive-coding • Signature

[ZeroJudge Link (Zh)(http://zerojudge.tw/ShowProblem?problemid=a003)

#include <stdio.h>

main() {
  int month, day;
  while( scanf("%d %d", &month, &day) == 2) {
    int res = (month * 2 + day) % 3;
    switch(res) {
    case 0:
      printf("普通\n");
      break;
    case 1:
      printf("吉\n");
      break;
    case 2:
      printf("大吉\n");
      break;
    }
  }
  return 0;
}
