ZeroJudge: a053 Sagit's 計分程式

Jul 30, 2016 • Jun 2, 2018 • WillyPillow • en, zh, bilingual, zerojudge, c, competitive-coding • Signature

ZeroJudge Link (Zh)

#include <stdio.h>
#include <math.h>

main() {
  int num, res;
  while (scanf("%d", &num) != EOF) {
    if (num >= 40) res = 100;
    else if (num > 20) res = 80 + (num - 20);
    else if (num > 10) res = 60 + (num - 10) * 2;
    else res = num * 6;
    printf("%d\n", res);
  }
  return 0;
}
