Jan 22, 2017 • Aug 22, 2018 • WillyPillow • Signature

The following are my GnuPG keys for signing code and this blog, email encryption, and so on.
The keys can also be downloaded in the keys/ directory.

Private Key Security

Different subkeys for different purposes are placed in separate networkless VMs in Qubes OS and are used via the Split GPG architecture. The master key is stored in a dedicated networkless vault VM.
