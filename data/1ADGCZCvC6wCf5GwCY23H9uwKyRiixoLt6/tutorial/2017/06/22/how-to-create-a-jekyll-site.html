How to Create a Jekyll Zite

Jun 22, 2017

  1. install ruby and add execution path to .bashrc: PATH="$(ruby -e 'print Gem.user_dir')/bin:$PATH". [ref] Then install Jekyll with $ gem install jekyll bundler

  2. Create a new site: $ jekyll new my-p2p-blog and a empty zite (From ZeroHello 3-dot menu)

  3. Edit /my-p2p-blog/_config Copy the new zite address to baseurl. Also edit other thing as you like. Newest json gem seems incompatible, so I add gem "json", "2.0.2" in /my-p2p-blog/Gemfile. [ref]

  4. Install gems $ bundle install, then update config with $ bundle exec jekyll build. Then copy files in /my-p2p-blog/_site to your new zite: $ cp /my-p2p-blog/_site/* /path/to/zeronet/data/ZITE_ADDRESS/.

  5. On your new zite, open the config panel (Drag to 0 button to left) and sign your content.json.
