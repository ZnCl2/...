How It Works

  • This site is javascript-free frontend for file hosts. Form on this page sends POST request directly to chosen file host, just like file host's website.
  • Your download link (or error message) is in raw JSON. Just read it and look for "full" URL. JSON is sent by file host, so this site cannot see result or format it.
  • This site does not store any files. Your files go to file host, not to here.

Example JSON

status: true
data:  
  file:  
    url:  
      full: "https://bayfiles.com/aBcD12eF34/Possible_Illegalities.7z"
      short: "https://bayfiles.com/aBcD12eF34"
    metadata:  
      id: "aBcD12eF34"
      name: "Possible_Illegalities.7z"
      size:  
        bytes: 67108864
        readable: "67.11 MB"

Security

  • Your file is not transferred through this site. Your browser connects to clearnet file host via Tor, sending directly to it. Only extra security you get by using this site is avoiding javascript. Otherwise it is exact same as visiting file host's site and uploading from there.
  • Once you click Upload, you are talking to file host, not to this site.
  • Web server logs timestamp when this page is accessed (like every web server). It keeps for only 48 hours. It does not know who you are, what you do on page, what you upload, where you upload, whether you even upload anything at all, etc.
  • This site does not send cookies.
  • Site does not contain ads, trackers, web bugs, scripts, iframes, fingerprinting, external stylesheets or images with codes in URL, etc. It is single self-contained HTML page with no external resources, not even images or favicon. You can view page source to verify this.

Misc

  • This site will remain available until file hosts start blocking anonymous uploads or until my hosting provider bans me because of site name.
  • You can upload from Linux without using browser. For example, to upload file "Possible_Illegalities.7z" to bayfiles.com, just do:

    torsocks -i curl -F file=@Possible_Illegalities.7z https://api.bayfiles.com/upload

    You can replace "bayfiles.com" in that URL with host of your choice from list above.
