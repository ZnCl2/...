What's Snakez?

Snakez is a python program to create HTML, CSS, JavaScript and PHP code fast and easily.

This framework has been created for people that know and love to code, and want to speed up their work,

In fact Snakez has a lot of built in code that you can use and personalize! Also you can easily modify the

Source code, to add your own template!

Snakez has a really simple and intuitive interface and it allow you to save all your template in your projects folder!

And remember : A terminal and a text editor is all that you need... WordPress is for pussies ; )


How to run Snakez :


shell ~ /Snakez $ python snakez.py

Functions

Snakez is still in beta phase but nevertheless it has al lot of functions.

The first function is a creator for basic HTML and CSS code that you need to start a new project;

This feature is really useful to start coding quickly.

The second function is more instresting, in fact is a collection of 5 different type of navigation bar

That you can customize according to your needs. In this collection you can find :


- A Simple menu bar. Example

- A Dropdown menu bar. Example

- A Sticky menu bar. Example

- A Responsive menu bar. Example

- A Responsive Sticky menu bar. Example


The third function is a responsive layout where you can put all your content without thinking about the responsive part of your page

Thanks to this function you have a grid responsive layout that allows to to create a responsive page quickly. Example

The last function in this beta phase is a collection of login and register forms that is can use in your personal website!

It use PHP and the access to your MySQL database, in fact it has all the instructions to configure your database with the template!

You can choose four different type of forms :


- A Simple Login from.

- A Simple Register & Login form.

- A Professional Login form.

- A Pofessional Register & Login form.


Have fun!

Sources


Snakez include jQuery and a different framework that is used for the sticky navigation bar.

In fact when you move the files created in the "YOUR_STUFF" folder, you must copy the "src" folder

Otherwise the template will not work properly.

Also the "src" folder contains some images that you need in your template.

If you lost your "src" folder and you don't want to re-download Snakez,

You can simply click here and download all the sources that you need.
