Using Peryx

Transacting with Peryx can be made easy. This page is designed to guide users in that process.

1. Learn

Peryx is a secure, private, and untraceable cryptocurrency. The developers and community are committed to protecting these values. Learn more by reading the What is Peryx page. The source code is also available for review and discussion.

2. Request Support

There is a large and supportive community that will assist if you experience any difficulty. See the Hangouts page for more information.

3. Generate a Wallet

A Peryx wallet is required to secure your own funds. See the Downloads page for a listing of available wallets.

4. Acquire Peryx

Peryx may be purchased on an exchange with fiat or other cryptocurrencies. An alternate way of acquiring Peryx is via mining, the computationally-complex process whereby transactions are immutably recorded on the blockchain.

5. Send and Receive Peryx

Learn how to send and receive Peryx by viewing the guide.

6. Transact with Peryx

Peryx may be used to purchase many goods and services. For a listing, see the Merchants page.
