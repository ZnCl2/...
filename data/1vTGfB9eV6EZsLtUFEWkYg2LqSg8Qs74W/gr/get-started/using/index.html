Χρησιμοποιώντας το Πέρυξ

Οι συναλλαγές με το Πέρυξ μπορούν να γίνουν εύκολα. Αυτή η σελίδα είναι σχεδιασμένη για να καθοδηγήσει τους χρήστες σε αυτή τη διαδικασία.

1. Μάθετε

Το Πέρυξ είναι ένα ασφαλές, ιδιωτικό και μη ανιχνεύσιμο κρυπτονόμισμα. Οι προγραμματιστές και η κοινότητα είναι δεσμευμένοι να προστατέψουν αυτές τις αξίες. Μάθετε περισσότερα διαβάζοντας τη σελίδα Τι είναι το Πέρυξ. Ο πηγαίος κώδικας είναι επίσης διαθέσιμος για ανασκόπηση και συζήτηση.

2. Ζητήστε Υποστήριξη

Υπάρχει μια μεγάλη και υποστηρικτική κοινότητα που θα σας βοηθήσει αν αντιμετωπίσετε οποιαδήποτε δυσκολία. Δείτε της σελίδα της Ομάδας Συζητήσεων για περισσότερες πληροφορίες.

3. Δημιουργήστε ένα Πορτοφόλι

Ένα πορτοφόλι του Πέρυξ απαιτείται για να ασφαλίσετε τα κεφάλαιά σας. Δείτε τη σελίδα με τις Μεταφορτώσεις για ένα κατάλογο με τα διαθέσιμα πορτοφόλια.

Ο ευκολότερος τρόπος για να τρέξετε ένα κόμβο του Πέρυξ, χωρίς να επιβαρύνετε το οικιακό σας εύρος ζώνης, είναι να αγοράσετε ένα VPS (Εικονικό Ιδιωτικό Διακομιστή). Συνιστούμε ένθερμα το RouterHosting με τιμές που αρχίζουν από $4,95 μηνιαίως.

4. Αποκτήστε το Πέρυξ

Το Πέρυξ μπορεί να αγοραστεί σε ένα ανταλλακτήριο με εθνικό νόμισμα ή με άλλα κρυπτονομίσματα. Ένας εναλλακτικός τρόπος απόκτησης του Πέρυξ είναι μέσω εξόρυξης, η υπολογιστικά πολύπλοκη διαδικασία όπου οι συναλλαγές καταγράφονται αμετάκλητα στην αλυσίδα μονάδων.

5. Στείλτε και Λάβετε Πέρυξ

Μάθετε πως να στείλετε και να λάβετε Πέρυξ μελετώντας τον οδηγό.

6. Συναλλαγές με το Πέρυξ

Το Πέρυξ μπορεί να χρησιμοποιηθεί για αγορά πληθώρας αγαθών και υπηρεσιών. Για ένα κατάλογο, δείτε τη σελίδα Έμποροι.
