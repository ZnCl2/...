Tutorial

(rebuild yourself)
all sources from above are linked exactly now and can be used for this.
please first check if your downloaded files are newer as the latest uploads. just use valid domains.
i will at colors to differ sources in ad and sec versions later.

1.
search for ' multiple reverse ip lookup ' to convert ip lists to domain names, this includes
sslbl ,feodotracker [their domainlist seems empty] and projecthoneypot

2.
replace a word in textfile=
sed 's/before/now/g' old_file > new_file
can be used for 127.0.0.1 <-> 0.0.0.0 (first isnt supported by all devices and second one reacts faster and needs less space)
malc0de source needs change of PRIMARY to ip,
for those without any hosts like formatation you can add before each line=
sed -e 's/^/yourtext/' old_file > new_file
[e.g. palevo on abuse.ch]
also to unify the distance between the ip and domain name in a host file, use these=
sed 's/127.0.0.1 /127.0.0.1 /g' old_file > new_file
[for moab and malwaredomainlist hosts]
sed 's/127.0.0.1\t/127.0.0.1 /g' old_file > new_file
[hosts-file.net , zeustracker and adzhost]
sed 's/127.0.0.1\t /127.0.0.1 /g' old_file > new_file
[for sysctl]


3.
merge host files into one
cat file1> tmphost
cat file2>> tmphost
and remove dubplicates
cat tmphost|sort|uniq >done1


all cmd commands works in mass - u can create a txt file with all actions together, copy&paste will let cmd work trough.
thats how i manage it. every version i update the source files ,edit each name with an date addition and just change the same at this txt file.

About

About

Inspired by BSDgeek_jane's mother of adblock, ive combined more than 19 hostname lists into one.
Its name should speak itself- this will keep most dangerous sites out of your range, and ads should be gone too.
The focus stays on security/privacy. cause the huge file could slow down your android device, two mobile versions got published. at the beginning, the moab was a mid-way cause its trimmed down already to take care of this point but now all should work fine.
btw you can also find tips to improve your device on this page.
!!
-- Important --
Firefox seems to ignore hosts file, import settings to IceCat or Pale Moon (modified Firefox)
this is big security bug, known by devs since about 8 years and still no change
!!

starting with android =

disable some services/activities on google apps (play services 'SystemUpdateService' ....) - battery
search smaller alternatives to default apps (quickpic...) - performance
check default performance options in settings
use tweaks (Fly-On Mod, Malive Booster, flag_tuner, sched_tuner, skeleton, maybe nitrox and still the best: supercharger)
