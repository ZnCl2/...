Carregando ...
 Jogue sem Cartas


       Gosta das cartas TCG, mas não tem grana para fazer um deck perfeito ou não tem ninguém na sua cidade que saiba o que é uma carta treinador? Então conheça o Apprentice. Com ele você poderá fazer sua lutas do TCG pela internet, e o melhor, não gastar nenhum centavo em cartas!
  

  
 Jogando com o Apprentice

Por BillySky


Ingredientes
       Para começar você irá precisar de :
 :::: Install do Apprentice
 :::: Patch para Pokémon TCG
 :::: Theme para Pokémon TCG
 :::: Conhecimentos de manipulação de arquivos no Windows



Começando
  ::::  Primeiramente, instale o Apprentice. Esse programa, feito pela Dragon Studios, foi originalmente feito para o jogo de Magic The Gathering, portanto suas configurações originais, o que incluem cards, decks e temas são todos virado para o Magic. Para que seja possível jogar Pokémon, é necessário que sejam utilizados alguns patches, e que seu adversário tenha os mesmos patches no computador dele.

 :::: Pegue o Patch para Pokémon TCG. Esse patch deve conter os seguintes arquivos, no mínimo :
 :: Addon.dat
 :: Distro.dat
 :: Expan.dat
 :: Format.dat
 :: Rules.dat
 :: cardinfo.dat
      Para nossos usuários aconselhamos esse Patch. Existem outros Patch na internet, alguns com cartas mais recentes, mas nós recomendamos esse por ser o melhor.

 :::: Todos esses arquivos, com exceção do cardinfo.dat, deve ser colocado na pasta em que você instalou o Apprentice. O arquivo cardinfo.dat deve ser colocado na subpasta 'Sets' do diretório no qual você instalou o Apprentice. Caso seja perguntado se você quer substituir os arquivos relacionados, responda 'sim'.

 :::: Pegue o Theme para Pokémon TCG e coloque-o, de preferência, na subpasta 'graphix' do diretório no qual o Apprentice foi instalado. O theme usado nesse guia pode ser adquirido aqui.

 :::: Você está pronto para rodar o Apprentice pela primeira vez! Rode o programa (não estranhe caso ele demore um pouco para carregar, ele é meio lerdo mesmo) e prepare-se para as primeiras configurações dentro dele!


No programa
  :::: Vá no menu Options, e escolha Set Name. Digite um nome ou um nick de sua preferência.

 :::: Vá no menu Options novamente, e escolha preferences. Na aba General, as 5 opções são, respectivamente : Bips ativados, Bips originários de um arquivo de som específico, Anexar a Grade (os cards são atraídos por uma grade invisível para facilitar a organização), Aparecer Dicas e Modo Caos. Como padrão, deixe marcado as 4 primeiras opções.

 :::: Vá para a aba Themes. Escolha change, e escolha o arquivo que você colocou na pasta graphix após ter descompactado o Theme para Pokémon TCG. Você verá que, agora, o seu apprentice terá uma aparência de um tablado de Pokémon TCG. Após isso, dê OK.


Antes...


... e depois do Theme instalado!

 :::: Vamos verificar se a instalação do patch dos cards foi feita corretamente! Vá no menu File e escolha Deck Editor. Caso a instalação tenha sido bem sucedida, você deverá ver cards de pokémon no espaço inferior esquerdo. O espaço à esquerda são os cards disponíveis, o espaço à direita é o seu deck atual. Para modificar o nome de seu deck, clique em 'Untitled Deck' e renomeie-o.


A janela de montagem dos decks

 :::: Os 6 botões entre os cards disponíveis e o seu deck são utilizados para visualizar e manipular a montagem de seu deck.
 :: View Card - visualizar card
 :: Comment - adicionar comentários ao seu deck
 :: Add - escolha o card respectivo à esquerda e clique em Add para adicioná-lo ao seu deck.
 :: Input - não é utilizado por pokémon, serve para adicionar um card que não esteja na lista
 :: Sideboard - não é utilizado por pokémon, refere-se a uma característica do Magic
 :: Delete - escolha o card que você queira retirar à direita e clique em delete para removê-lo.

 :::: Com isso, monte os seus decks da melhor forma que você quiser. Mas, caso você não queira ficar com fama de ladrão, obedeça as regras de montagem de deck de Pokémon TCG. Para salvar os seus decks, clique em 'save' lá em cima e escolha um nome de arquivo.

 :::: Caso você queira carregar um deck previamente salvo, clique em load e escolha o arquivo. Caso você queira começar um deck novo, clique em new. Para salvar o seu deck em um arquivo .txt, clique em text. 'p De' serve para imprimir o deck atual, 'p Li' serve para imprimir a lista de cards total. Go e Stop serve para caso você queira fazer uma pesquisa pelos cards existentes.

 :::: Com isso, vamos para um primeiro jogo!


Primeiro jogo
  :::: Vá no menu 'file', escolha 'new game'. Nesta janela, clique em Load Deck para escolher um deck previamente feito. Após isso, em 'game options' coloque Starting Life como 0 e Starting Cards in Hand, 7. Em Starting Message você pode colocar algum grito de guerra, ou frase preferida. Para esse primeiro jogo, escolha em 'game type' Play a Solitaire Game, e dê OK.


Janela para Novo Jogo!


 :::: Este primeiro jogo não será contra ninguém, servirá apenas para você se acostumar com os comandos. Vale lembrar que o Apprentice não possui opção 'contra o computador', portanto o jogo deverá ser feito sempre sozinho ou então conta um oponente pela internet.

 :::: De acordo com as configurações feitas na tela anterior, sua mão agora deve possuir 7 cards. Os 3 botões acima dos cards que estão na sua mão serão bem utilizados, com exceção do 3º, que não será utilizado. O primeiro representa a pilha de compra - arraste esse botão para o espaço dos cards da sua mão (my hand) para comprar uma carta, dê um duplo clique nesse botão para embaralhar o seu deck (library). O segundo representa a pilha de descarte - arraste uma carta de sua mão para ele para descartar uma carta, dê um duplo clique nele para visualizar a lista de cartas já descartadas.


O botão à esquerda e o central são usados, o da direita não

 :::: É hora de colocar os prêmios na mesa. Para colocar cards na mesa, é só clicar no primeiro botão e arrastar para a mesa. No entanto, isso coloca um card à mostra na mesa - para colocá-lo de cabeça pra baixo, segure shift antes de proceder como acima. Coloque-os no respectivo espaço (prizes), lembrando que geralmente são 6 os prêmios.

 :::: É hora de começar o jogo! Verifique se você tem algum pokémon básico na mão. Caso você tenha, arraste-o (virado) para a mesa, no lado selecionado. Caso você tenha mais algum que queira deixar no banco, faça o mesmo procedimento e coloque-o no banco. Não o vire ainda! Caso você não tenha nenhum pokémon básico, vá no menu Action, escolha Hand e a opção 'Move Hand into Library'. Após isso, dê um duplo clique no primeiro botão acima do espaço reservado para a sua mão e arraste 7 cards do seu deck para a sua mão. Enquanto isso, o oponente compra 2 cards caso ele queira. 

 :::: Com os pokémons posicionados, é hora de ver quem começa! Escolha quem quer cara, e quem quer coroa. Escolha quem vai virar a moeda também. Para virar uma moeda, vá em Action e escolha 'Flip a Coin'.

 :::: Agora, todos viram os seus pokémons para o lado certo e jogo comece. Antes de mais nada, lembre-se que este jogo é completamente manual, portanto vocÊ pode fazer tudo, inclusive coisas ilegais - no entanto, o oponente estará vendo tudo o que você está fazendo, portanto você estará sempre sendo observado. Uma atitude não autorizada deve ser imediatamente explicada ou então vocÊ poderá ser taxado de ladrão =P

 :::: Aquele que começar o jogo compra uma carta, e procede normalmente. Para atachar uma energia ao seu pokémon, arraste uma energia da sua mão para próximo ao pokémon escolhido. Caso você queira utilizar uma trainer, coloque a trainer na mesa e siga suas instruções (para visualizar o que qualquer card faz, clique com o botão direito e escolha view card). Obedeça as instruções do card!

 :::: Ao clicar com o botão direito em um card seu, vocÊ possui uma infinidade de opções que podem ser utilizadas para que as regras do jogo sejam obedecidas. São elas :
 :: Tap - pode ser utilizado para marcar um card paralizado, envenenado, confuso ou dormindo.
 :: Attack, Attack Without Tapping e Doesn't Untap as Normal - não são usados
 :: Phase Out - Também Não
 :: Flip card over - virar o card ao contrário (ou vice-versa)
 :: Color lace - Muda o fundo dos cards - não é utilizado
 :: Set counters - modifica o número de marcadores da carta. Geralmente, para 10 de dano, considera-se 1 counter, assim como no jogo real.
 :: Add counter - adiciona um marcador
 :: Remove counter - remove um marcador
 :: Switch controller - o card muda de dono (passa ao controle do adversário)
 :: Bury - descarta a carta
 :: Return to hand - a carta volta pra mão
 :: Return to library - a carta volta pro deck
 :: Remove from game - não é utilizado
 :: View card - ver as características.
Veja que qualquer card que volta pro deck volta para o topo, portanto o deck deve ser posteriormente embaralhado!


O menu de um card

 :::: Após terminar seu turno, você deve atacar. Anuncie qual é o seu ataque via chat e calcule o dano - o adversário deverá colocar os counters respectivos e fazer qualquer ação respectiva à seus movimentos e ações, e vice-versa.

 :::: Lembre-se - basicamente tudo é manual nesse jogo! Portanto, cooperação dos dois lados é básico para que haja um jogo honesto e fluente.

 :::: O jogo acaba quando um dos oponentes compra todos os seus prêmios, o deck do adversário acaba ou então o adversário não possui mais pokémons em mesa, como no pokémon real. É isso!


Jogando online
 :::: Para você jogar online, os passos são os mesmos - no entanto, na janela de new game, escolha 'connect via the internet' e dê OK.


Jogando pela Net!

 :::: Em todo jogo deve existir um servidor e um cliente. Isso deve ser previamente discutido entre os jogadores, via ICQ ou algum outro programa de conversação instantânea de preferência. O servidor, na janela que aparecer após dar OK no 'connect via the internet', deve escolher 'waiting for a call' e dar ok.

 :::: Esse mesmo servidor deverá ver que, no campo 'My Current IP', aparecerá um número. Esse é o seu IP, ou seja, o número que te representa na internet quando você está conectado. Este número deve ser informado para o cliente que, ao escolher ok na janela de New Game, possa selecionar 'calling my opponent at' e colocar o número corretamente na janela e dar ok.

 :::: ATENÇÃO ! CUIDADO !
O IP é um número que te identifica na internet. Isso significa que, da mesma forma que a pessoa pode se conectar a você para jogar um jogo online, ela pode utilizar programas maus para invadir o seu computador apenas especificando o seu IP. Portanto, cuidado ao fornecer esse número! Tenha sempre um anti-vírus atualizado para manter-se livre de trojans e um bom firewall para proteger-se de conexões alheias. A PokeLand não se responsabiliza por qualquer dano que venha a acontecer ao seu computador devido à má distribuição e utilização deste número por sua parte ou qualquer outra coisa referente.

 :::: Problemas com IP : As vezes o tipo de conexão da pessoa não permite que o usuário seja servidor do jogo, por isso quando você não conseguir receber nenhuma ligação, faça com que o seu adversário abra a conexão.

 :::: Após conectado, o jogo deve fluir normalmente, como já explicado.


FAQ - Perguntas Freqüentes
P: Meu apprentice não mostra os cards de pokémon! O que devo fazer?
R: Siga corretamente os passos referentes à colocação de arquivos na pasta do apprentice.

P: Eu já verifiquei! O que eu faço?
R: Verifique de novo. O problema é lá. Quer apostar?

P: Quero.
R: Então verifica de novo, pô!

P: Oops. Desculpa.
R: Por nada.

P: Pô, ainda assim não consegui ver os cards!
R: Então não sei. E não me torra.

P: Não consigo jogar pela internet!
R: Confira o número do IP para ver se está correto. Verifique também se o oponente está 'waiting for a call' (esperando por uma chamada)

P: Onde eu consigo os cards com o texto em português?
R: Perdão, mas não existe nenhum patch em português até o momento.

P: Mas eu não sei ler inglês!
R: Nem eu japonês.

P: Mas você consegue jogar Apprentice!
R: Pois é.

P: Minha janela de chat sumiu! O que eu faço?
R: Vá no menu options, escolha preferences e dê OK - deve voltar ao normal.

P: Meu oponente está roubando! O que eu faço?
R: Converse com ele. Caso ele continue roubando, saia do jogo e mantenha provas disso.

P: Onde eu posso salvar um log?
R: Menu File, 'Save Window Message Buffer'.

P: É tudo manual? Como fazer caso aconteça _____________ ?
R: Existem algumas situações em que você precisa adaptar para o Apprentice. No entanto, alguns cards que não possuem como - estes cards estão marcados com a palavra 'UNPLAYABLE'. Nestes casos, não há saída.

P: Meu pc foi invadido por causa do IP que eu falei! Vocês me pagam!
R: A gente avisou. Não é culpa nossa.

P: Para que servem todas as outras opções do menu do Apprentice?
R: Você pode explorar isso. Caso você saiba um pouco de inglês, será mais fácil. Para facilitar, o menu Game não será utilizado, com exceção de 'Clear Buffer' (limpar chat). Nem o menu help nem o references serão usados também. O menu 'communications' em princípio não precisa ser usado, mas pode auxiliar.

P: O card tal aparece na tela, mas ele não possui nenhuma explicação de ataque ou Pokemon Power!
R: Provavelmente deve haver alguma diferença nos patches. Conversem entre si para verificar qual é o patch mais atualizado e, o que necessitar, atualize-se.
