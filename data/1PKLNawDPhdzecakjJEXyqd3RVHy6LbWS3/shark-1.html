 Usando o GameShark


    Popular entre os jogadores que querem tudo mais fácil, jogadores de todo o mundo acabam sendo atraídos pelos resultados rápidos que esta ferramenta oferece. Usar ou não usar esses métodos ilícitos de utilização de jogos vai da ética de cada um.
 
Usando e Comprando e GameShark Por JoJo

     Uma das mais populares ferramentas para diversos vídeos games, esta permite que através de um código específico você tenha acesso a recursos do jogo facilmente. Um exemplo no Pokémon é a capacidade de pegar Pokémons que seriam impossíveis para a versão adquirida, ou aquele que é impossível de pegar usando apenas o jogo (Ex. Celebi).

Aviso!
Antes de colocar qualquer código salve o seu jogo, pois existe o risco que você perca todos os seu dados ou mesmo que o jogo trave na hora de colocar o código. Recomenda-se moderação na utilização dos códigos.

      Existem diversas formas de utiliza-lo, clique nos links abaixo para conhecer seu método de utilização:

  • Usando no emulador
  • Usando no Game Boy / Game Boy Advance / SP
Usando no Emulador Por JoJo

     Primeiramente é necessário saber se seu emulador tem opção de colocar códigos de GameShark. Em nossa seção de emuladores são listados emuladores que possuem esse característica. Como avaliamos que o emuladore VisualBoyAdvance é o melhor para emular jogos de Game Boy em geral, a seguir teremos um guia de como utilizar um código.
 

Passo 1 :: Abra a lista de truques

No VisualBoyAdvance, carregue sua ROM. Após a abertura da ROM, só assim o menu de "Truques" ("Cheats" na versão americana) será habilitado. Clique no Truques, e acesse Lista de Truques (Cheat List) conforme a imagem ao lado.
 
Passo 2 :: Adicione um código de GameShark

Irá ser exibida uma janela conforme a imagem ao lado. Nela clique no botão Gameshark para abrir a tela de inserção de novo código GameShark conforme a imagem abaixo. Nela coloque a Descrição conforme o seu gosto, e no campo código coloque o código necessário para ativar o truque. Caso o truque precise de mais de um código conforme a tela abaixo, digite cada um espaço de quebra de linha entre eles. Após confirmar no botão OK será adicionado em sua lista o código do truque. Será possível depois Remover (Remove) e também apenas desativar um truque. Dê OK para confirmar os truques, e volte ao jogo.

Pronto seu código já deverá estar funcionando!

 
 

Usando no Game Boy / Game Boy Advance / SP Por JoJo

      Para usar o GameShark em sua fita de GB/GBA é necessário comprar um dispositivo que é uma ponte na ligação do console e o cartucho. Depois de ter ligado este aparelho que aparenta um cartucho, será aberta a opção de inserir o código, que poderá estar incluído numa biblioteca própria que a maioria destes dispositivos possuem. O uso é simples, mas comprar que é complicado. No Brasil não encontrei nenhuma loja na Internet que vendia desse dispositivo, nem mesmo nos leilões. Normalmente as pessoas acabam comprando este através de um importador, que trás ele especialmente para você.

     Para comprar, no caso para GB/GBA existem dois produtos similares: GameShark (o original) e o Action Replay (a cópia). Ambos executam códigos GameShark em seu jogo, a diferença geralmente está no preço, e nas características de biblioteca interna. Segue abaixo imagens das capas dos dois dispositivos.
