 Muda a Data / Hora


      Esse guia é o método para você trocar a data do relógio do jogo. Ele foi feito para apenas funcionários da Nintendo saberem o password para quando o usuário tiver mesmo que trocar o horário (mudança de fuso horário por exemplo). Mas com esse guia você poderá trocar de dias e de hora para pegar Pokémons ou Itens que só estão disponíveis em determinado momento.

      Porque você gostaria de mudar o horário? Para encontar pessoas, Pokémons e eventos! Para fazer seus Pokémons fazerem breeding, ou para o ovo chocar, mudar o relógio não irá alterar isso pois o que conta nesses casos é o número de passos que você dá.

      Atenção :: Esse código só vale para as versões Gold / Silver, no Crystal não existe um método sem o GameShark para mudar o horário do jogo.

 

  
 Muda a Data / Hora do Gold / Silver

Por JoJo

      Para colocar o password no seu jogo, fique apertando as seguintes teclas: BAIXO + SELECT + B logo depois que você ligar o jogo com o cartucho ligado. Você irá ver o logo da Nintendo e logo depois, se você apertou as teclas, ele irá perguntar se você quer mesmo trocar, coloque YES (para sim). A seguinte tela irá aparecer para colocar a senha (password)

                            

     Para saber qual é a sua senha, você poderá usar dois métodos: automático e manual.
 
Método Automático
     Simplesmente coloque os dados necessários nos campos abaixo. Atenção, para que o formulário abaixo funcione, o Scrips Ativos (JavaScript) terão que estar ativados em seu navegador. Caso o código gerado não funcione, tente depositar seu dinheiro com sua mãe para tentar gerar um novo código.

Ajuda
  :::: Nome do treinador é o nome que está lá no menu do jogo, você deverá colocar seu nome como você o registrou, por isso, o nome "JoJoao" será diferente de "jojoao". Caso você colocou o PK , coloque ao invés { , para MN o coloque }, e para × o troque por *.
  :::: ID é o número único que você recebe, para saber qual é o seu ID, va no item que tem o seu nome no menu e lá você irá encontrar um ID de 5 dígitos.
  :::: Dinheiro é quanto você tem "no bolso" não contando o que a sua mãe está guardando. Para saber quando você tem, acesse o menu com o seu nome e lá você vai encontrar seu Money. Coloque apenas números, e se você não tiver dinheiro simplesmente coloque "0".
                              


    Ao colocar sua senha, e apertar A, ele irá dar o OK e irá resetar o jogo, e ao entrar no jogo você irá encontrar um "TIME NOT SET" e ele irá levar você para uma tela onde você poderá colocar o dia da semana e a hora que você desejar.

Passo 2
Seu Dinheiro

     Como você pode ver na imagem eu tenho 3000. Pegue esse valor e divida por 256, e com o resultado some com o resto.

Se você tiver 3000
Dividindo por 256: 3000 / 256 = 11 , com resto 184
Soma o resultado com o resto: 11 + 184 = 195

Caso esse número fosse maior que 256 (no meu caso não passou), você teria que pegar esse valor e faze o mesmo processo acima até que você encontre como resultado um número menor ou igual a 256

     PASSO2 = 195


Passo 3
Seu ID

     Faça o mesmo método que você fez com o dinheiro, dividindo por 256.

Meu ID é 46392
Dividindo 46392 por 256 : 181 com resto 56
Somando : 181 + 56 = 237

     PASSO3 = 237
