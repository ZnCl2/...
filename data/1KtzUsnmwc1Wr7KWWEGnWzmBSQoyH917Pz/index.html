KOPI

Boilerplate for ZeroNet web applications

Clone and code, don't repeat yourself

Create new ZeroNet projects cloning this site instead of starting from scratch, you'll get the basic ZeroNet site's directory structure, default json files and a base coffeescript file so you can start coding straight away.

Getting started

  • Go to your Hello ZeroNet page and look for Kopi in your connected sites list.
  • Click on the three dots button and clone the site.
  • Open the newly cloned site's folder and start coding using the files provided by Kopi.
