IPFS MMDB

IPFS Multi-Media Database Project

IPFS DOES NOT SUPPORT TOR YET BUT, A VPN WILL HIDE YOUR IP FROM THE NETWORK

Submit IPFS hashes to music, movies, and shows!

I'm going to start development on a distributed entertainment website using ZeroNet and IPFS to host content. First, we need the content, that's where you come in. Download your favorite movies, music and TV shows and host them on IPFS, when the site is finished other users will be encouraged to pin files and host them as well to maintain persistent storage.

For more info, or help with IPFS, click here

Entry Format Example:

Media Type, Title, Artist/Director, Genre, Album, Year

Music, Dinan, Oskar Schuster, Classical, Sneeuwland, 2014, /ipfs/QmcYKHpcBJNL9uaxxTZJGuk9QYBfS3cDuYKoELPhfdmRqn

Be sure to check the "More Info" area for more guidelines on submitting

Notes:

IPFS seems to best handle .mp4 and .mp3 files. .mkv gets corrupted, .flac and .wmv can't be streamed, only downloaded. You can easily convert files using VLC. I will post a script later to help people with a lot of files to convert

For media type please only use Music, Movie, or Show

For genre please just keep it simple, Classical, Classic Rock, Country, Metal, Comedy, Action, Drama, etc.

If submitting an album, please put "NA" as the title and put the title of the album where it belongs

If submitting a movie, please put "NA" under album

If submitting a show, please put the season number under album

Movie hashes MUST be direct to the movie, and shouldn't contain actual words. If your hash does, run "ipfs reslove /ipfs/hash/whatever"

For now, all hashes are going to auto-convert to direct links. In the future the list will be prettier and the links will take you to a media player that allows playlisting
